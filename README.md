### ⚠️ Troubleshooting

**Common issues and solutions**

#### Issue: Python command not recognized

```powershell
# Solution: Add Python to PATH or use full pathpy script.py  # Alternative command on Windows
```

#### Issue: Module not found

```powershell
# Solution: Install the required modulepip install module_name
```

#### Issue: Permission denied when installing packages

```powershell
# Solution: Use --user flagpip install --user package_name
```

#### Issue: Multiple Python versions

```powershell
# Use specific versionpy -3.12 script.py  # Python 3.12py -3.11 script.py  # Python 3.11
```
